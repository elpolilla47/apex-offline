# Apex

Aplicacion instalable de metas personales, finanzas y calculadoras.
Funciona offline despues de abrirla una vez con internet.

## Instalar en Android

1. Abrir el enlace de GitHub Pages en Chrome.
2. Tocar el menu de tres puntos.
3. Elegir **Instalar aplicacion** o **Agregar a pantalla principal**.
4. Abrir Apex desde el icono del telefono.

Los datos quedan guardados solamente en el dispositivo.
